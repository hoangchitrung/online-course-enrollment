# online-course-enrollment
### A. Course Selection:
- Users register with educational background and preferred language.
- Courses are categorized as "Self-Paced" (Video-based) or "Live Boot Camp."
- Self-Paced has 10 modules; Live Boot Camp has 8 scheduled weekly sessions.
- Users select their preferred start cohort (Month/Year).
### B. Course Cart:
- Users can bundle multiple courses for a discount.
- Users can upgrade from Self-Paced to Live sessions in the cart.
- Confirmed enrollments move to "My Learning," and a syllabus PDF is exported.
